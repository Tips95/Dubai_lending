# Настройки для Timeweb Cloud

## Важно! Настройки в панели Timeweb Cloud:

### Если используете Docker:

1. **Тип приложения**: Docker
2. **Dockerfile**: автоматически найдет в корне
3. **Порт контейнера**: `3000`
4. **Переменные окружения**:
   ```
   NODE_ENV=production
   PORT=3000
   HOSTNAME=0.0.0.0
   ```

### Если используете Next.js (без Docker):

1. **Тип приложения**: Next.js
2. **Версия Node.js**: 20
3. **Команда сборки**: `npm run build`
4. **Команда запуска**: `npm run start` (уже настроена на 0.0.0.0)
5. **Порт**: `3000`
6. **Переменные окружения**:
   ```
   NODE_ENV=production
   PORT=3000
   HOSTNAME=0.0.0.0
   ```

## Проверка после деплоя:

1. Откройте логи приложения в панели Timeweb
2. Проверьте что видно:
   ```
   ▲ Next.js 14.0.0
   - Local:        http://0.0.0.0:3000
   - Ready in X ms
   ```
3. Если видите ошибки - скопируйте их

## Если всё ещё 404:

Проверьте:
- ✅ Сервер запустился (есть сообщение "Ready")
- ✅ Слушает на 0.0.0.0:3000 (не localhost)
- ✅ Нет ошибок в логах
- ✅ Порт в настройках Timeweb = 3000
