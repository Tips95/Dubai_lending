# Развертывание на Timeweb Cloud

## Настройки в панели Timeweb Cloud:

1. **Тип приложения**: Next.js
2. **Версия Node.js**: 20
3. **Команда сборки**: `npm run build`
4. **Команда запуска**: `npm run start`
5. **Порт**: 3000 (или автоматически определяемый)
6. **Директория сборки**: `.` (корень проекта)

## Важные настройки:

- Убедитесь что `package.json` находится в корне репозитория
- Проверьте что все зависимости установлены
- Убедитесь что порт настроен правильно

## Если возникает 404:

1. Проверьте логи развертывания в панели Timeweb
2. Убедитесь что сборка прошла успешно
3. Проверьте что команда `npm run start` запускается
4. Проверьте настройки порта

## Переменные окружения (если нужны):

В панели Timeweb Cloud можно добавить:
- `NODE_ENV=production`
- `PORT=3000` (если требуется)
