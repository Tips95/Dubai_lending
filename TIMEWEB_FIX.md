# Решение проблемы 404 на Timeweb Cloud

## Проверьте настройки в панели Timeweb Cloud:

### 1. Тип приложения
- Выберите: **Next.js** (НЕ Docker, если есть выбор)
- Или если только Docker: убедитесь что Dockerfile правильный

### 2. Команды
- **Сборка**: `npm run build`
- **Запуск**: `npm run start`
- **Порт**: `3000` (или автоматический)

### 3. Переменные окружения
Добавьте:
```
NODE_ENV=production
PORT=3000
```

### 4. Если используете Docker
Убедитесь что:
- Dockerfile находится в корне проекта
- Команда запуска: `npm start`
- Порт: 3000

## Возможные проблемы:

1. **Неправильный порт** - проверьте логи, на каком порту запустился сервер
2. **Проблемы с роутингом** - убедитесь что используется Next.js режим, а не статический экспорт
3. **Кэш** - попробуйте пересобрать приложение заново

## Что проверить в логах:

После развертывания проверьте логи:
- Успешно ли прошла сборка (`npm run build`)
- Запустился ли сервер (`npm start`)
- На каком порту работает приложение
- Есть ли ошибки при запуске
